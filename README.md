# AI Task Analytics Dashboard

## Project Information
The AI Task Analytics Dashboard is designed to provide insights and analytics on various tasks processed through an AI engine. It helps in monitoring performance, tracking efficiency, and managing workloads effectively through a user-friendly interface.

## Tech Stack
- **Frontend:** React.js, Redux, HTML5, CSS3
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Hosting:** Heroku / AWS
- **Additional Libraries:** Chart.js for data visualization, Axios for API calls

## Features
- **Real-Time Analytics:** Live updates on task performance and metrics.
- **User Authentication:** Secure login functionality.
- **Data Visualization:** Interactive charts and graphs to display analytics data.
- **Task Management:** Features to create, update, and delete tasks.
- **User Profiles:** Allows users to manage their profiles and preferences.
